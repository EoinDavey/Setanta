## Gníomhartha

| Ainm | Cur síos | Sampla |
|----
| `scríobh` | Scríobh luach ar an gconsól | `scríobh('Dia duit')` |
| `ceist` | Scríobh téacs ar an gconsól agus léigh an freagair | `ceist('Cén aois thú?')` |
| `léigh` | Léigh téacs ón gconsól | `léigh()` |

## Actions

| Name | Description | Example |
|----
| `scríobh` | Write a value on the console | `scríobh('Dia duit')` |
| `léigh` | Read text from the console | `léigh()` |
| `ceist` | Write text on the console and read the answer | `ceist('Cén aois thú?')` |

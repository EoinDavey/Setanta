# Conas Setanta a úsáid

Chun triail a bhaint as Setanta, tá dhá rogha agat.

- Is féidir leat Setanta a úsáid ar [try-setanta.ie](https://try-setanta.ie).
- Íoslódáil é ó NPM le `npm i -g setanta`.
